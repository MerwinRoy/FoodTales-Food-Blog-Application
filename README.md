# CSYE 7280 - User Experience Design and Testing (FINAL PROJECT)
## FoodTales: Food Blog Application

![Image post-bro](https://user-images.githubusercontent.com/46862684/207488870-92a6a7f5-fc97-4cf4-b1ad-179af3189ad9.svg)

## Project Description
Created user-centered designs using Figma design tool for FoodTales which is a food blogging mobile application by implementing interactive,visual design interface and user experience best practices for a seamless,intuitive and cohesive user experience.

## Project Objectives
Designed a food blogging platform that assists bloggers and people with passion for food and/or photography to exhibit their skills, learn new recipes, whilst also acting as a point of reliable reference for cuisine/restaurant reviews and explore the culinary world.

## Design Process
  - User surveys
  - Defining Problem Statement, User Personas, Comparative Analysis
  - Brainstorm Sessions, Card Sorting, Information Architecture
  - Wireframing and Interactive Prototyping using Figma
  - Usability Testing and Prototype Testing
